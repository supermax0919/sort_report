氣泡排序法 (Bubble Sort)

原理： 從頭開始比較相鄰的兩個元素，如果前面的比後面大就交換位置。每一輪結束後，最大的數字會像「氣泡」一樣浮到最後面。

C 語言實作碼


#include <stdio.h>

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n-1; i++) {
        for (int j = 0; j < n-i-1; j++) {
            if (arr[j] > arr[j+1]) {
                // 交換數值
                int temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
}

int main() {
    int data[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(data)/sizeof(data[0]);
    bubbleSort(data, n);
    printf("排序後的結果: \n");
    for (int i=0; i < n; i++) printf("%d ", data[i]);
    return 0;
}

時間複雜度分析

  • 最差情況：$O(n^2)$
  • 最佳情況：$O(n)$ (已排序時)
